# Sled Test Analyzer (SPUL)

Bu sürümde uygulama yalnızca SPUL modülüyle çalışacak şekilde sadeleştirildi. Kapak ve Photo Report modülleri kaldırıldı.

## Kurulum

```bash
python -m pip install -r requirements.txt
```

## Çalıştırma

```bash
python app.py
```

> Not: Uygulama açılışında eksik paketleri otomatik kurmayı dener. Ağ/pip kısıtı varsa yukarıdaki kurulum komutunu manuel çalıştırın.
